scss用法

1.安装ruby

2.命令行：gem install sass

3.sass四种编译风格

3-1 nested：嵌套缩进的css代码，它是默认值

3-2 expanded：没有缩进的、扩展的css代码

3-3 compact：简洁格式的css代码

3-4 compressed：压缩后的css代码

4.生产环境：sass --style compressed test.sass test.css

5.让SASS监听某个文件或目录，一旦源文件有变动，就自动生成编译后的版本

5-1 sass --watch input.scss:output.css

5-2 sass --watch app/sass:public/stylesheets

6.导入编译后的css文件即可
